# AGENTS.md

## Project Overview
Buddhi AI is an advanced AI-powered in-browser AI.

## Core Technologies
- Next.js
- Shadcn UI
- Tailwind CSS
- Mediapipe
- PGlite
- LlamaIndex TypeScript
- Gemma 3n

## Setup & Build Commands
- Install dependencies: `pnpm install`
- Start dev server: `pnpm dev`
- Build: `pnpm build`
- Start production server: `pnpm start`
- Lint: `pnpm lint`

## Instructions
- **Understand the Project**: Familiarize yourself with the Buddhilive AI project.
- **Code Quality**: Ensure that all code suggestions adhere to best practices for readability, maintainability, and performance.
- **Feature Implementation**: When suggesting code for new features, ensure they align with the overall architecture and design patterns of the project.
- **Bug Fixes**: When suggesting fixes for bugs, provide clear explanations of the changes made and why they resolve the issue.
- **Error Handling**: Ensure that all code suggestions include appropriate error handling and edge case considerations. Handle potential exceptions gracefully.
- **UI Design**: Always make sure the UI match the current theme of the application and always add Tailwind style classes for light mode and dark mode.
- **Package Manager**: Use PNPM as the package manager.

## Don'ts
- Avoid suggesting code that introduces security vulnerabilities.
- Do not suggest code that deviates from the established coding standards of the project.
- Refrain from suggesting overly complex solutions when simpler alternatives exist.
- Do not generate implementation documentation or comments unless specifically requested.
